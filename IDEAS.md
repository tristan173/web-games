# Web Games — Idea Catalog

Goal: intuitive, addictive, fun. Single self-contained `.html` files — vanilla JS + Canvas, zero dependencies, zero build. Destined for web-game portals (CrazyGames / Poki class) and/or GitHub Pages. A little extra $ if one or more hit.

House style (set by Ping): near-black blue field, one cold accent + one warm accent, mono uppercase type, tiny corner HUD, tiny WebAudio synth. "The delight should be visible but the rigor should be load-bearing."
